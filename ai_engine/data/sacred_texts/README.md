# Sacred / public-domain sources (placeholders)

Place UTF-8 `.txt` files here (one per work or per book). The build script chunks them for `sacred_qa.jsonl`.

Suggested **public domain** sources (you must download copies yourself due to size):

| Work | Notes |
|------|--------|
| Bible (KJV) | Project Gutenberg |
| Bhagavad Gita | e.g. Gutenberg etext |
| Tao Te Ching | Gutenberg / Wikisource |
| Dhammapada | Gutenberg |

The repo may ship **tiny excerpts** only for pipeline testing; replace with full texts on your training machine.

Do not commit multi-megabyte files unless your repo policy allows it.
